# grok-high-conviction-calls

**AI-Powered Weekly High-Conviction Options Advisor**

A simple, transparent tool that uses Grok + strict rules to recommend **one high-quality long call (or rare put)** per week for beginner traders.

### How It Works
- Runs every Sunday night
- Analyzes 12 assets using 4 competing AI agents
- Only recommends a trade if it passes **8 strict filters**
- Gives you exact Robinhood instructions (ticker, strike, expiration, limit prices)
- Automatically tracks how past recommendations would have performed

### Tech Stack
- Next.js + Vercel
- Supabase (database + cron)
- Grok API (xAI)

### Philosophy
Quality over quantity. We skip most weeks. When we trade, we want high conviction only.

Built for beginners who want a disciplined, rules-based approach without having to watch the market all day.
